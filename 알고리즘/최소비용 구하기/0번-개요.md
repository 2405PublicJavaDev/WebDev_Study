# 개요
## 문서 관리자
조승효(문서 생성자)
## 시작
   - 다익스트라 알고리즘
      - 간선의 비용이 0이상일 때에만 사용해야 한다.
      - 간선의 비용이 음수면 사용할 수 없다.
      - 출발점 도착점이 하나일 때 쓰기 딱 좋다.
   - 플루이드 워셜 알고리즘
      - 그래프에서 가능한 모든 노드 쌍에 대해 최단 거리를 구하는 알고리즘
      - 시간복잡도는 O(V^3)
      - 다익스트라 알고리즘과는 달리 모든 노드 쌍에 대해 최단 거리를 구하고, 음의 가중치를 가지는 그래프에서도 쓸 수 있다는 것이 특징이다.
